# #️⃣ Locked-In

A premium, minimalist student planner web application built for deep focus and frictionless organization. 

**For live demo visit :- https://rashij-17.github.io/Locked-In/)**

---

## ⛰️ Vision & Aesthetic

**Locked-In** is designed to counteract the hyper-stimulating, neon-drenched productivity apps of the modern web. It features a low-saturation, high-contrast, distraction-free interface inspired by high-end analog stationery. No harsh glows, no vibrant distractions—just pure focus.

---

## 🎨 Core Features

*   **Intelligent Dashboard:** A high-level daily overview featuring an elegant radial clock and key productivity metrics.
*   **Chronological Agenda:** A fluid, interactive timeline powered by smooth spring physics animations.
*   **Immersive Focus Mode:** A breathing Pomodoro timer with subtle, calming background color transitions to anchor your attention.
*   **Dynamic Tagging:** Effortlessly categorize tasks by subject, project, or priority level.
*   **Tailored Themes:** Shift your environment instantly with premium, low-stimulus presets: *Sand, Slate, Paper, Forest,* and *Dusk*.
*   **Offline-First Architecture:** Instant local storage caching guarantees zero load lag, paired with optional Firebase/Supabase cloud synchronization.

---

## 🛠️ Tech Stack & Architecture

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Framework** | Next.js 14 (App Router) | Structural skeleton and static optimization |
| **State** | Zustand | Lightweight, high-performance global client state |
| **Styling** | Custom Low-Sat CSS | Unified design tokens for elite minimalism |
| **Auth & Sync** | Firebase Auth / Supabase | Secure user authentication and optional cloud sync |
| **Deployment** | GitHub Pages | Fully static `out` folder distribution |
