## ⛰️ Vision & Aesthetic

**Locked-In** exists to counteract the hyper-stimulating, notification-drenched productivity apps of the modern web.

It's built on a philosophy borrowed from high-end analog stationery — low-saturation palettes, high-contrast typography, and generous negative space. Every interaction is deliberate. Every color earns its place. The result is an interface that disappears so your work doesn't have to.

> *If your planner is louder than your thoughts, it's not helping you focus — it's competing with you.*

## 🎨 Core Features

| | |
|---|---|
| 🕰️ **Intelligent Dashboard** | A radial analog clock paired with live task counts, focus time, and completion stats — your day, at a glance. |
| 📋 **Chronological Agenda** | A fluid, interactive timeline driven by smooth spring-physics animations. |
| 🎯 **Immersive Focus Mode** | A breathing Pomodoro timer with calming, subtle background transitions to anchor your attention. |
| 🏷️ **Dynamic Tagging** | Categorize tasks by subject, project, or priority — organize your way. |
| 🌗 **Tailored Themes** | Five low-stimulus presets — *Sand, Slate, Paper, Forest,* and *Dusk* — to match your mood and light. |
| ⚡ **Offline-First Architecture** | Instant local caching for zero load lag, with optional Firebase/Supabase cloud sync. |

## 🚀 Getting Started

### Prerequisites

- **Node.js** v18 or higher

### Local Development

```bash
# Clone the repository
git clone https://github.com/Rashij-17/Locked-In.git
cd Locked-In

# Install dependencies
npm install

# Run the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

### Build for Production

```bash
npm run build
```

This generates a fully static `out/` folder, ready for deployment to GitHub Pages or any static host.
