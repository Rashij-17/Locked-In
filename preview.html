✨ Core Blueprint Features

  • Radial Clock Layout: High-precision geometric 24-hour SVG matrix mapping active daily items as clean, low-saturation progress arcs.
  • Breathing Focus Space: Concentric countdown timer driven by weighted spring mechanics and micro-pulses to govern rest intervals.
  • Chronological Horizon: Fluid visual timeline sorting tasks into Today, Tomorrow, and Upcoming priority lanes.
  • Identity Gateways: Clean, minimalist route guards integrated directly with client-side Firebase Authentication.
  • Dual Sync Engine: Latency-free local caching managed via Zustand, backed by structural streams to a secure cloud Supabase relational core.

🚀 Terminal Command Pipeline

Initialize Variables (.env.local in root):

NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_id

NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

Local Engineering Environment Initialization:

git clone https://github.com/Rashij-17/Locked-In.git
cd Locked-In
npm install
npm run dev
